Noosphere

Working memory for your AI tools.

Your thinking is scattered across threads, tools, and devices. Each conversation is a silo — what you figured out in one doesn't exist in the next. Noosphere connects them.

Think out loud on your phone, build at your desk, refine in a different tool. They all share the same brain.

Connect your tools

Claude.ai web

1. Go to claude.ai/settings → Integrations / MCP Servers
2. Add server URL: https://usenoosphere.ai/mcp
3. Complete the OAuth login

Claude Code

claude mcp add noosphere --transport http --scope user https://usenoosphere.ai/mcp

Any other MCP tool

Point it at https://usenoosphere.ai/mcp

Then just talk naturally

“What were we working on?”
“Remember this decision”
“Save where we are”

The AI handles everything behind the scenes. No commands to learn.
